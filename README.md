# 📈 Stock Trading System (v97)

RSI + 移動平均 + 押し目戦略による  
日本株のシンプルな自動売買システムです。

---

## 🎯 戦略概要

- RSI + MA + 押し目
- 保有期間：4 日（固定）
- 資金管理：70%運用（v97）
- 銘柄数：1 ポジション（集中型）
- 少額スタート（例：20000 円）

---

## ⚙️ エントリー条件

以下すべて満たす場合：

- 株価 > MA(25)
- 前日比で押し目（pullback）
- RSI < 65
- 出来高条件クリア

---

## 🚪 イグジット条件

- **4 日後に強制決済（time exit）**

---

## 💰 資金管理

- 1 トレードで資金の 70%を使用
- 残り 30%はキャッシュ
- 複利で増加

---

## 📊 バックテスト結果（概要）

| pullback | 評価             |
| -------- | ---------------- |
| 0.002    | 安定             |
| 0.004    | **最適（推奨）** |
| 0.007    | 攻撃的           |
