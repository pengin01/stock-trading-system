# v90 Daily Trading System

## 概要

本プロジェクトは、日本株を対象とした**日次トレードシステム**です。  
GitHub Actions により毎日自動実行され、Discord へシグナルを通知します。

v90 では以下を実装しています：

- 資金管理（ポジションサイズ自動計算）
- 複数ポジション対応
- ポジション自動更新
- Discord 通知

---

## 戦略概要

### エントリー条件

- 株価 > 移動平均（MA）
- 軽い押し目（前日比 -0.5%）
- RSI が閾値以下（デフォルト 65）
- 出来高条件（VALUE20）

---

### エグジット条件

- MA 割れ
- 保有日数経過（デフォルト 4 日）

---

## 資金管理（v90 の特徴）

- 初期資金：80,000 円
- 1 トレードリスク：2%
- 最大ポジション数：3
